# Personal Portfolio & Résumé

A single-page personal portfolio and résumé website for a Finance professional & self-taught Software Engineer — 1× Founder.

## Stack

- **HTML5** — semantic markup
- **CSS3** — custom properties, CSS Grid/Flexbox, animations
- **Vanilla JavaScript** — IntersectionObserver, scroll effects, form handling

## Sections

| Section | Description |
|---|---|
| Hero | Name, title, key badges, CTA buttons |
| About | Bio with stats (finance years, engineering years, founder) |
| Skills | Finance & engineering skill bars with animated fills |
| Experience | Career timeline (Founder → Freelance → Analyst → Junior Analyst) |
| Ventures | Project cards including featured fintech startup |
| Education | BCom Finance, self-taught CS, CFA Level I |
| Contact | Social links + contact form |

## Running locally

```bash
# No build step — just open the file
open index.html
# or
npx serve .
```
